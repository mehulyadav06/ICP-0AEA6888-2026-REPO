# AI Chatbot using Gemini API

## Features

- Conversational chatbot
- Context memory
- Error handling
- Gemini API integration

## Technologies

- Python
- Google Gemini API(For free)
    1. Go to:
    Google AI Studio(https://aistudio.google.com/api-keys)⁠
    2. Login with Google account.
    3. Click Get API Key.
    4. Click Create API Key.
    5. Copy the key.
