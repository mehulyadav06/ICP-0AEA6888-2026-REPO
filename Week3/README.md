# Gemini AI Chatbot

## Description

A conversational AI chatbot built using Google's Gemini API.

## Features

- Real-time conversation
- Conversation memory
- Error handling
- Exit command
- User-friendly interface

## Technologies Used

- Python
- Google Gemini API

## Installation

1. Clone repository
2. Install dependencies
    pip install -r google-generativeai
3. Add Gemini API Key
4. Run chatbot
  python chatbot.py

## Sample Commands
- What is AI?
- Explain Python loops
- Give me interview tips

Type "exit" to close chatbot.
