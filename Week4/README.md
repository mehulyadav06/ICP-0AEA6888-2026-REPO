# AI Content Summarizer

## Overview

This project uses OpenAI GPT models to summarize
text and PDF documents.

## Features

- Text Summarization
- PDF Summarization
- Multiple Summary Lengths
- Streamlit UI
- OpenAI API Integration

## Tech Stack

- Python
- OpenAI API
- Streamlit
- PyPDF2

## Installation

pip install -r requirements.txt

## Run

streamlit run app.py
